## min

minimalist serverless messenger working over webrtc

[gh-pages demo](https://k-olya.github.io/min/)


Install dependencies
```
npm i
```

### Available scripts

Run in dev mode
```
npm run dev
```
or
```
npm start
```


Generate a production build
```
npm run build
```


Run eslint
```
npm run lint
```


Vite preview
```
npm run preview
```
