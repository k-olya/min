## React + TypeScript + Tailwind with Vite

This is a starter project for react/ts single page applications using tailwindcss styles.
Install dependencies
```
npm i
```

### Available scripts

Run in dev mode
```
npm run dev
```
or
```
npm start
```


Generate a production build
```
npm run build
```


Run eslint
```
npm run lint
```


Vite preview
```
npm run preview
```
